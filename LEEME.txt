Generador de Proformas - App MySQL
Requisitos
- Python 3.8+
- Cliente de MySQL en PATH (comando: mysql)
- Servidor MySQL (root, puerto 3306; o configura config.json)

Estructura
coteind_app/
├─ app.py
├─ db.py
├─ config.json
└─ sql/
   ├─ schema_seed.sql        (esquema + datos iniciales)
   ├─ procedures_all.sql     (procedimientos/funciones)
   └─ triggers.sql           (triggers)

Cómo ejecutar
1) Verifica que “mysql” funcione en consola (ej. `mysql --version`).
2) Abre `config.json` y ajusta host/port/user/password/database si es necesario.
3) Ejecuta: `python app.py`
4) En la ventana de conexión, “Probar conexión” y luego “Guardar y continuar”.
5) La app ejecutará automáticamente:
   - schema_seed.sql (solo si la tabla Empleado no tiene datos)
   - procedures_all.sql
   - triggers.sql

Login
- Ingresa el Código de empleado con formato EXXXX (solo números en la caja; la “E” se completa sola).
  Ejemplo: 0001 → E0001

Solución de problemas:
- “No se encontró ‘mysql’…”: agrega la carpeta del cliente MySQL al PATH (ej.: C:\Program Files\MySQL\MySQL Server X.Y\bin).
- No detecta la database: será necesario modificar el config.json, con el localhost, port, user, password, database.

Revisar Manual de usuario para mayor información.
